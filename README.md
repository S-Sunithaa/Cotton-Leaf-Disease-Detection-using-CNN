# Cotton Leaf Disease Detection using CNN

## Project Overview
This project is a deep learning based system that detects diseases in cotton leaves using a Convolutional Neural Network (CNN). 
The model analyzes images of cotton leaves and predicts whether the leaf is healthy or affected by a disease.

## Objective
The main objective of this project is to help farmers and agricultural researchers identify cotton leaf diseases quickly using image classification.

## Technologies Used
- Python
- Convolutional Neural Networks (CNN)
- Deep Learning
- Django (for web interface)
- TensorFlow / Keras
- HTML, CSS
- OpenCV

## How It Works
1. User uploads an image of a cotton leaf.
2. The image is processed using the CNN model.
3. The model predicts the disease type.
4. The system displays the detected disease result.

## Features
- Image based disease detection
- Deep learning model for classification
- Web interface for easy use
- Accurate prediction using CNN
